## 一、阿瓦倫對話

### 1. 自動說話（AVAREN_DLG）

階段判斷函數 `getAvarenDlg()`：
- 心情 ≥ 80 → **占有**
- 信任 ≥ 60 或 心情 ≥ 60 → **開放**
- 信任 ≥ 30 或 心情 ≥ 30 → **溫和**
- 其他 → **冷漠**

觸發時機：不定時冒泡泡（間隔 200～500 幀隨機），以及互動選單「說說話」。

```
（對阿瓦倫）嘿！一起玩嗎——（阿瓦倫走開了）（雷德原地站著）……好，我懂了。
（看阿瓦倫坐在那裡）他在想什麼？……（想了很久）……我問不出來，算了。
（小聲）那孩子雖然不理我，但感覺他一直在觀察我……（挺起胸膛）我要表現得更像勇者！
他是在保護你嗎？每次我靠近你，他眼睛就移過來……這孩子，其實很在意你嘛。
（偷偷在地上放了一朵花）（看阿瓦倫）（阿瓦倫踩過去了）……好，這方法不行。
（偷偷看阿瓦倫）……他今天好像比昨天靠你近一點？我有眼花嗎。
（對艾德林，小聲）你有沒有教他怎麼跟人說話？我可以去當他的練習對象！……（看阿瓦倫的眼神）……算了可能不行。
（看阿瓦倫）……他現在是在看我，還是看我背後的東西？（往旁邊移了一步）……還是在看我。
（小聲）我帶了一個小零食，想給那孩子，但他……怎麼感覺會把我的手咬掉……
（認真）那孩子的眼神，讓我想到一隻受傷的小動物。（停頓）……我決定不靠近了，讓他慢慢來。
（看著阿瓦倫跟在艾德林後面）……哇，他跟你走得好近。上次他還躲你後面，現在不一樣了啊！
```

## 三、旁白文字（NARR 系列，各 10 句）

### 採集藥草（NARR_GATHER）
```
你在田地裡採了一把藥草。回到屋子時，阿瓦倫靜靜坐在窗邊，沒有看你——他等了很久。
草葉上還帶著露水。你採著採著，想起阿瓦倫說的某句話，不由自主地停了下來。
你蹲在田邊採藥，遠遠能看見小屋的窗光。不知道他現在在做什麼。
藥草的氣味有些苦。你把它們裝進袋子，心想：應該夠了。如果夠的話。
你採完藥回來，阿瓦倫沒有說話，但你注意到他的視線在你手上的藥草停了很久。
今天的藥草長得比平常好。你多採了一些，回去的路上心情輕了一點。
採藥的時候，一隻蟲子爬上了你的手背。你停下來看它走了一會兒，才繼續工作。
風把藥草的香氣吹散了，整片田都帶著一股淡淡的苦味。你覺得這個氣味最近聞起來安心了許多。
你的手沾著泥土回來，阿瓦倫看了一眼，沉默地拿來了一條布巾放在你旁邊。
今天採的藥草有一把特別新鮮，你想著等會兒讓阿瓦倫看看——也許他會感興趣。
```

### 出門工作（NARR_WORK）
```
你到村子替人搬貨、打雜。傍晚回來時，阿瓦倫坐在角落，背對著你。你知道他又一個人待了很久。
有人問你那個孩子是誰。你說是弟弟。對方沒再追問，但你的心跳快了一拍。
你在市集搬了一整天的貨，腰很酸。回到小屋時，阿瓦倫還醒著，坐在黑暗裡，沒有開燈。
你拿著錢走回來。天黑了，阿瓦倫一個人在那間小屋裡。你不由自主地加快了腳步。
村子裡有人好奇地打量你。你低頭繼續幹活，心裡想的是：再快一點，早點回去。
你推著木輪車走了一整個下午，手上磨出了薄繭。回家的路，走得比平常快。
有個老婆婆多給了你幾個銅板，說你是個勤快的孩子。你謝了她，走回去的時候想：這些夠買食物了。
工作的時候開始下起小雨。你加快速度，滿腦子想的是小屋裡的窗戶有沒有關。
傍晚收工，你把賺來的錢數了一遍，剛剛好夠這幾天用。你握著那些銅板，覺得稍微安心了一點。
今天工作時，有個路過的小孩盯著你看了很久。你突然想起了阿瓦倫第一次見到你時的眼神，加快腳步回家。
```

### 烹飪（NARR_COOK）
```
你在小廚房裡仔細烹調了一道簡單的料理。熱氣騰騰的香味飄進了房間，你聽見阿瓦倫輕輕靠近了門口。
你試著做了阿瓦倫好像喜歡的味道。端出去的時候他沒說好吃，但盤子空了。
食材不多，你想了很久才決定怎麼做。熱氣從鍋子裡升起來，讓這間小屋有了一點生氣。
你把飯端進去，阿瓦倫看了一眼，沒說話。但他拿起了筷子——這樣就夠了。
鍋子燒焦了一點邊。你端出去之前猶豫了一下，最後還是端出去了。阿瓦倫沒有評論，只是吃完了。
你切菜的聲音在小屋裡迴響。沒多久，阿瓦倫悄悄站在廚房門口，靜靜地看著你——然後走開了。
你想讓食物口感軟一些，多燉了一會兒。端出來的時候，阿瓦倫難得用力多吃了幾口。
你在鍋裡加了一點你猜他可能喜歡的調味。他吃完之後停頓了一下，然後把碗放好，沒有說話。
今天食材很少，你搭配了又搭配，才拼出了一頓說得過去的飯。阿瓦倫吃完了，你才終於也坐下來吃自己那份。
熱氣從鍋子裡散出來，讓整間小屋都暖了起來。你想，哪怕只有這樣，也比什麼都沒有好。
```

### 研究藥水（NARR_RESEARCH）
```
你翻閱古卷，嘗試調配解咒的藥水。阿瓦倫身上的黑霧稍微淡了一些，像是有什麼鬆動了。
你在燭光下讀著晦澀的古文，眼睛開始發酸。詛咒的根很深，但你感覺找到了一點線索。
藥水的顏色不對，你重做了一次。窗外天色漸暗，你沒有察覺。
你將配好的藥水靠近阿瓦倫，黑霧微微收縮。他沒有說謝謝，但他讓你靠近了。
古卷上有幾行字你完全看不懂。你把它們抄下來，告訴自己明天再試。
你把幾種藥草磨碎，試著照著古卷上的步驟調配。混合之後的氣味有些嗆，但形狀是對的。
燭火搖曳，你用手遮著光繼續讀。這些字句寫得很深奧，像是故意不讓人輕易看懂。
你調配了一整個下午，最終得到了一小瓶勉強看起來正確的藥水。你把它放好，長吐了一口氣。
書頁碎了一角，你小心地把它拼回原位。這本書比你還要懂得阿瓦倫身上的詛咒。
你把調好的藥水拿給阿瓦倫看。他盯著那瓶子看了一會兒，伸出手，讓你幫他用。
```

### 說說話（NARR_TALK）
```
你輕聲和阿瓦倫說了幾句話。他沒有回應太多，但你能感覺到他稍微放鬆了。
你說了一些今天的事。阿瓦倫沒有看你，但他在聽。
你靠近阿瓦倫，說了幾句不重要的話。沉默比言語還多，但感覺比什麼都不說好一些。
你問他今天怎麼樣。他沒有回答，只是把膝蓋抱得更緊了一點。你就這樣坐在旁邊。
你說了一個小故事。阿瓦倫沒有任何反應——但他沒有叫你停下來。
你說了幾句話，沉默了一下，又繼續說。阿瓦倫的呼吸漸漸平穩，肩膀不再那麼繃緊。
你告訴他今天田裡看見了什麼。說到一半他偏了偏頭，像是在想那是什麼樣子。
你說話的聲音很輕，怕驚擾了什麼。阿瓦倫沒有離開，你覺得這樣就夠了。
你問了他一個很小的問題。他猶豫了很久，最後用手指在地上輕輕點了一下——算是回答了。
你說著說著，發現阿瓦倫不知不覺移近了一些，靠著牆坐，距離你只差一個手臂的長度。
```

### 聊天陪伴（NARR_CHAT）
```
你坐在阿瓦倫身旁陪著他。時間慢慢流逝，但你覺得，這樣的時間是值得的。
你們說了很多，也有很長的沉默。最後他說了一句你沒想到的話，你不知道該怎麼回，只是笑了笑。
阿瓦倫今天說的話比平常多了一點點。你不敢表現得太明顯，怕嚇到他。
你陪著他坐了很久，說些有的沒的。窗外的光慢慢暗下去，你們誰都沒有起身。
他說了一件很小的事，像是不經意說出來的。你後來想了很久，覺得那也許不是不經意的。
你問了他一個關於以前的問題。他沉默很久，最後說了幾個字，然後就不說了——但說出來了，就是說出來了。
阿瓦倫今天主動說了一件事。你沒有立刻回應，讓那句話在空氣裡停了一下，再輕聲接話。
你們並排坐在窗邊。說了很多，也什麼都沒說。你覺得這樣的安靜，比任何話都要踏實。
他講了一個你完全不懂的東西，說了半天又說「算了，你不會知道的」。但你讓他繼續說完了。
你說著說著，忽然發現阿瓦倫在笑。不是那種大笑，是嘴角很輕、很快閃過去的那種。你沒有指出來。
```

### 讀書給阿瓦倫聽（NARR_READ）
```
你從書架取下一本舊圖鑑，翻到植物那一頁，輕聲讀給阿瓦倫聽……
你念著書上關於星星的文字，一字一字解釋給阿瓦倫聽……
你拿出一張手繪地圖，告訴阿瓦倫這片森林叫什麼名字……
你翻到一個關於動物的小故事，開始輕聲念給阿瓦倫聽……
你找出一本藥草辨識的書，指著圖解釋每一種植物的用途……
你翻出一本很舊的故事書，封面有點破損。你輕輕撫平折角，從第一頁開始念……
你找到一段描述遠方城市的文字，讀著讀著，想知道阿瓦倫有沒有去過那些地方……
你把書放在兩個人中間，一邊念一邊用手指著對應的圖，讓阿瓦倫也能看清楚……
這本書有點難，你遇到不懂的字就停下來想一下再繼續。阿瓦倫靜靜等著你找到答案……
你念到書裡一個小孩的故事，頓了一下，繼續念下去，沒有說任何多餘的話……
```

### 照顧換藥（NARR_CARE）
```
你幫阿瓦倫仔細清理傷口，換上乾淨的紗布。他默默地讓你做，你心裡的愧疚稍微淡了一點。
阿瓦倫的傷口比昨天好了一些，但詛咒留下的痕跡還在。你輕輕包紮，盡量不讓他覺得疼。
你幫他換藥的時候，他一直看著別的地方。你問他痛不痛，他沒有回答。但他沒有躲開。
換完藥之後，阿瓦倫輕輕說了一聲「……謝謝」。你愣了一下，他已經把臉轉開了。
他身上的詛咒今天比較安靜。你幫他整理頭髮，他沒有拒絕，只是沉默地坐著。
你用溫熱的布替他擦了臉。他瞇起眼睛，沒有說話，但肩膀鬆開了。
詛咒紋路今天蔓延得比昨天寬。你沒有說出來，只是更仔細地幫他清理，告訴自己會好的。
你輕輕掀開他的袖子，看見新的傷痕。你什麼都沒說，安靜地包紮好，把袖子放回原位。
他今天讓你幫他換藥的時間，比上次長了一點。你沒有問原因，就這樣慢慢地做完。
你換完藥，準備起身時，阿瓦倫的手輕輕碰了一下你的袖口，然後縮回去。你裝作沒看見，但心跳快了一拍。
```
